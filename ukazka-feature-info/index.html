Feature info

Angular komponenta obrazovky zamčené funkce ve Fidoo — šablona a SCSS styly. Jen ukázka, kód na stránce nic nespouští.

Šablona (HTML)

<div class="content">
    @if (showSidebar) {
        <app-sidebar-close autoopen (close)="close()"></app-sidebar-close>
    }

    <div class="header">
        <d-badge [color]="'accent'">
            <span translate>
                {{
                    featureData.badgeTextKey ||
                        'feature.' + feature + '.lock.badge'
                }}</span
            >
        </d-badge>
        <div class="feature-actions">
            @if (customActionsRef) {
                <ng-container
                    *ngTemplateOutlet="customActionsRef"></ng-container>
            } @else {
                <d-button
                    (click)="openMoreAboutPage()"
                    [color]="showPriceList ? undefined : 'primary'">
                    <span translate>feature.{{ feature }}.lock.moreAbout</span>
                </d-button>
                @if (showPriceList) {
                    <d-button (click)="openPricingPage()" [color]="'primary'">
                        <span translate>fidoo.products.priceList</span>
                    </d-button>
                }
            }
        </div>

        <div class="heading">
            <h1 class="ta--left" translate>
                {{
                    featureData.titleKey || 'feature.' + feature + '.lock.title'
                }}
            </h1>
            <div class="description ta--left">
                <p translate>
                    {{
                        featureData.subtitleKey ||
                            'feature.' + feature + '.lock.subtitle'
                    }}
                </p>
                @if (showInfo) {
                    @if (isChampion$ | async) {
                        <p
                            [innerHTML]="
                                'feature.lock.info.champion' | translate
                            "></p>
                    } @else {
                        <p translate>feature.lock.info.regular</p>
                    }
                }
            </div>
        </div>
    </div>

    <div class="feature-previews">
        @if (featureData.images) {
            @for (
                image of featureData.images[currentLang || 'cs'];
                track trackByImageSrc($index, image)
            ) {
                <img [src]="image.src" alt="Feature preview" />
            }
        }
    </div>

    <div class="feature-cards">
        @for (
            info of featureData.info;
            track trackByFeatureName($index, info)
        ) {
            <div class="feature-card">
                <span class="icon-circle mb--1-2">
                    <i class="fal fa-{{ info.icon }}"></i>
                </span>
                <h3 class="mb--1-2" translate>
                    {{ info.titleKey }}
                </h3>
                <p translate>
                    {{ info.descriptionKey }}
                </p>
            </div>
        }
    </div>
</div>
